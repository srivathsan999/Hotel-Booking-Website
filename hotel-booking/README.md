# Elysian Hotel — Booking Website

## Overview
Production-ready, hospitality-grade hotel booking website built with HTML5, Tailwind CSS, and vanilla JavaScript.

## Structure
```
hotel-booking/
├── index.html
├── rooms.html
├── room-details.html
├── about.html
├── contact.html
├── login.html
├── signup.html
├── 404.html
│
├── assets/
│   ├── css/
│   │   └── tailwind.css
│   ├── js/
│   │   └── main.js
│   ├── images/
│   │   ├── rooms/
│   │   ├── gallery/
│   │   └── icons/
│   └── fonts/
```

## Run
- Open `index.html` in a modern browser.
- No build step required; Tailwind is loaded via CDN.

## Features
- Calm luxury aesthetic
- Sticky glass navbar
- Glass booking form
- Neomorphic primary CTAs
- Dark mode toggle with persistence
- Equal-height cards and clean responsiveness
